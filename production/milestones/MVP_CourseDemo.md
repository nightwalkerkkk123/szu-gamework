# Milestone: MVP 课程演示

## 目标

在课程答辩前交付可玩的 2D 横版滑雪跑酷垂直切片，验证「机制转译」设计方法在糖尿病健康教育游戏中的可行性。

## 范围

### 必含（MVP）

- 单人模式
- 1–3 关：L1 医院雪屋（必做）、L2 高血糖区（强推）、L3 低血糖区（弹性）
- 3 个核心道具：胰岛素喷雾、降糖药瓶、高糖雪花
- 1–2 个视觉区
- 血糖条、距离、暂停、结算 UI
- 跳跃 / 翻滚 / 滑行 / 躲避核心操作
- 60 fps 稳定运行

### 不含（扩展规划）

- 多人模式
- 新手引导模块
- 控糖成就系统
- 医疗百科图鉴
- 排行榜
- 每日挑战

## 成功标准

1.  playable：任意玩家可在 30 秒内理解操作
2.  educational：试玩玩家 ≥ 70% 正确回答 5 道问卷中的 4 题
3.  polished：无崩溃、帧率稳定、视觉风格统一
4.  demonstrable：5 分钟 Demo 能讲清机制转译
